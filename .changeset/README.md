# Changesets

This folder has been automatically generated by [`@changesets/cli`](https://github.com/changesets/changesets).

For each change that should ship in a release, add a changeset by running `npm run changeset` and committing the generated file with your PR. On merge to `main`, the Release workflow opens (or updates) a "Version Packages" PR that bumps the version and updates `CHANGELOG.md`. Merging that PR publishes to npm.

See the [common questions](https://github.com/changesets/changesets/blob/main/docs/common-questions.md) for more detail.
